# BSA11 - Проектирование пользовательских интерфейсов

## Описание проекта

Проект посвящён проектированию пользовательского взаимодействия и экранных форм для системы онлайн-записи в барбершоп. В рамках проекта я описал форму представления продукта, роли и их ключевые сценарии, состав экранных форм, данные и контроли ввода, а также разработал сами экранные формы в Balsamiq и draw.io, чтобы передать дизайнеру полную информацию для создания удобных и полезных интерфейсов.

---

## Что было сделано

- `ex00.docx`  
  Описаны форм-фактор, тип приложения, способы управления, потребности локализации.

- `ex01.docx`, `ex01.png`
  Составлены классификаторы и справочники с макетами экранных форм, описаны операции, права доступа и обеспечивающие функции (поиск, фильтрация, сортировка, логирование).

- `ex02.docx`  
  Определены поля экранных форм справочников, форматы, обязательность, дефолты, проверки и сообщения об ошибках, потребности пагинации и агрегатов.

- `ex03.docx`  
  Разработаны ключевые сценарии ролей с основными и альтернативными путями, схемами задач и переходов.

- `ex04.docx`  
  Составлен список экранных форм с идентификаторами, назначением, данными и правами доступа.

- `ex05.docx`, `ex05.png`
   Разработаны экранные формы всех экранов с описанием операций, поддерживающих функций, данных и логирования.

- `ex06.docx`  
  Описаны контроли ввода/корректировки для всех экранов, включая условия, сообщения об ошибках, результаты и параметры логирования.
